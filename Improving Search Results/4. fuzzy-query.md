# `fuzzy` query

There is a significant difference between match quesry with fuziness and 
fuzzy query

fuzzy query is a term lever query - so no analyzers are runned.
Only editing for term

```
GET /products/_search
{
  "query": {
    "fuzzy": {
      "name": {
        "value": "LOBSTER",
        "fuzziness": "auto"
      }
    }
  }
}
```

```
GET /products/_search
{
  "query": {
    "fuzzy": {
      "name": {
        "value": "lobster",
        "fuzziness": "auto"
      }
    }
  }
}
```
