# Querying parent by child documents

Return parent based on child criteria
https://www.elastic.co/guide/en/elasticsearch/reference/current/query-dsl-has-child-query.html#_sorting

score_mode - 5 score modes
- min max lowest/highest score of matching child doc is mapped into parent
- sum the matching children's scores are summed up and mapped into the parent
- the avg score based on matching child documents is mapped into the parent
- none Default =  scores of child docs are ignored

We can specify the min number of childs must be matched to parent doc be included
in search results
min_children
max_children

## Finding parents with child documents matching a `bool` query

```
GET /department/_search
{
  "query": {
    "has_child": {
      "type": "employee",
      "query": {
        "bool": {
          "must": [
            {
              "range": {
                "age": {
                  "gte": 50
                }
              }
            }
          ],
          "should": [
            {
              "term": {
                "gender.keyword": "M"
              }
            }
          ]
        }
      }
    }
  }
}
```

## Taking relevance scores into account with `score_mode`



```
GET /department/_search
{
  "query": {
    "has_child": {
      "type": "employee",
      "score_mode": "sum",
      "query": {
        "bool": {
          "must": [

                "age": {
                  "gte": 50

              }
            }

            {
              "term": {
                "gender.keyword": "M"
              }
            }

  }
}
```

## Specifying the minimum and maximum number of children

```
GET /department/_search
{
  "query": {
    "has_child": {
      "type": "employee",
      "score_mode": "sum",
      "min_children": 2,
      "max_children": 5,
      "query": {
        "bool": {
          "must": [
            {
              "range": {
                "age": {
                  "gte": 50
                }
              }
            }
          ],
          "should": [
            {
              "term": {
                "gender.keyword": "M"
              }
            }
          ]
        }
      }
    }
  }
}
```
